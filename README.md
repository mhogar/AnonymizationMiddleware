# Anonymization Middleware

## Dependencies
- [Node.js](https://nodejs.org/en/)

## Setup and Running

### Server
- `cd` into server
- run `npm install` (one time setup)
- run `npm run start` to start the backend server

### Client
- `cd` into client
- run `npm install` (one time setup)
- run `npm run server` to start the frontend development server
